Interpretao do IMC (faixas de referncia)

  • Abaixo do peso: IMC < 18.5
  • Peso normal: 18.5 <= IMC < 24.9
  • Sobrepeso: 25 <= IMC < 29.9
  • Obesidade grau 1: 30 <= IMC < 34.9
  • Obesidade grau 2: 35 <= IMC < 39.9
  • Obesidade grau 3: IMC >= 40

# -----------------------------------------------------------------------------
# 0. Preparao dos Dados
# -----------------------------------------------------------------------------

# Cria uma cpia do DataFrame original (supondo que 'dados' j contenha a coluna 'Faixa_IMC'
# e 'Tempo_sono'; a coluna 'faixa_sono' ser criada on the fly para agrupar Tempo_sono)
dados_binned = dados.copy()

# Se necessrio, converte a coluna Faixa_IMC para categorical com a ordem desejada
desired_imc_order = [
    "Abaixo do peso",
    "Peso normal",
    "Sobrepeso",
    "Obesidade grau 1",
    "Obesidade grau 2",
    "Obesidade grau 3"
]
dados_binned['Faixa_IMC'] = pd.Categorical(
    dados_binned['Faixa_IMC'],
    categories=desired_imc_order,
    ordered=True
)

# -----------------------------------------------------------------------------
# 1. Agrupamento dos Dados usando as Regras "On The Fly" para Tempo_sono
# -----------------------------------------------------------------------------

# Aplica o binning on the fly para Tempo_sono conforme as regras:
# [0, 6)   -> Baixo; [6, 8) -> Mdio; [8, 12] ou maiores -> Alto.
# (Aqui usamos pd.cut para criar a categorizao temporria)
faixa_sono = pd.cut(
    dados_binned['Tempo_sono'],
    bins=[0, 6, 8, 12],
    labels=['Baixo', 'Mdio', 'Alto'],
    right=False
)

# Cria uma pivot table agrupando por Faixa_IMC e pela categorizao de Tempo_sono
pivot = dados_binned.groupby('Faixa_IMC')['Tempo_sono'].apply(
    lambda x: pd.cut(
        x, bins=[0, 6, 8, 12], labels=['Baixo', 'Mdio', 'Alto'], right=False
    ).value_counts()
).unstack(fill_value=0)

# Reindexa o pivot para garantir a ordem desejada, tanto para linhas quanto para colunas
pivot = pivot.reindex(desired_imc_order, fill_value=0)
pivot = pivot[['Baixo', 'Mdio', 'Alto']]

Boxplot:

Utilizamos o Plotly Express (px.box) para criar um boxplot em que o eixo x representa as categorias de Alimentao e o y mostra os valores de IMC. Assim, possvel comparar visualmente a disperso do IMC em cada grupo.

ndice de Saudabilidade:

Como a varivel Alimentao qualitativa, criamos um mapeamento para convert-la em uma varivel numrica. Quanto melhor a alimentao, maior o ndice atribudo (por exemplo, Muito saudvel 4).

Grfico de Disperso:

Com o px.scatter relacionamos a varivel contnua Indice_saudabilidade com o IMC. Assim, possvel verificar se existe alguma tendncia, por exemplo, se um ndice mais alto (alimentao melhor) est associado a um IMC menor ou maior.
